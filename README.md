# Valyu AI SDK Playground

Interactive playground for testing [@valyu/ai-sdk](https://www.npmjs.com/package/@valyu/ai-sdk) tools with Vercel AI SDK v5.

## Quick Start

```bash
pnpm install
pnpm dev
```

Add your API keys to `.env.local`:

```
VALYU_API_KEY=your-api-key
AI_GATEWAY_API_KEY=your-vercel-ai-gateway-key
```

## Features

- Test all Valyu search tools (web, finance, papers, bio, patents, SEC, economics, company research)
- Stream or generate mode
- Multiple model support (Gemini, GPT, Claude, Grok)
- Live code snippet generation
